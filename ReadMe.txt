1. Run Server.py to create a web socket
2. Server.py will ask for specific ip and port to be created on.
3. Once Socket is created successfully, run client.py on another machine
on the same network.
4. Connect to the specified IP and Port from step 2.
5. Type in a alphanumeric input and receive the message back in Caps.
6. On a different machine from the ones running client.py and server.py
open an internet browser.
7. goto the specified IP and port in the browser
8. Included is index.html, index.jpg, and Test.html to test HTTP capabilities.
